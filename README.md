# SQL_CLR_Assembly_Spatial_Reproject
Reproject and convert SQL Server spatial data types to Lat Long or GeoJSON. 

## Dependencies
This solution requires the Dotspatial.Projections nuget package.

## Installation
Build the solution.
Move the SQL_CLR_Assembly_Spatial_Reproject.dll and DotSpatial.Projections.dll to the relevant SQL Server.
Adapt and run the sql script included in the project Load_Assembly_Create_Functions.sql
